# 🚀 App Login - Spring Boot

Projeto desenvolvido com o objetivo de aprender e aplicar conceitos de backend utilizando **Spring Boot**, criando um sistema simples de autenticação com cadastro e login de usuários.

---

## 🧠 Tecnologias utilizadas

* Java 17
* Spring Boot
* Spring Data JPA
* MySQL
* Hibernate

---

## ⚙️ Funcionalidades

* Cadastro de usuário
* Login de usuário
* Conexão com banco de dados MySQL
* Estrutura em camadas (Controller, Service, Repository)

---

## 🗄️ Banco de dados

O projeto utiliza MySQL como banco de dados.

Crie o banco com o comando:

```sql
CREATE DATABASE applogin;
```

---

## 🔧 Configuração

Configure o acesso ao banco no projeto:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/applogin
spring.datasource.username=seu_usuario
spring.datasource.password=sua_senha
```

---

## ▶️ Como executar

1. Clone o repositório
2. Abra no IntelliJ
3. Configure o banco de dados
4. Execute a aplicação

---

## 🎯 Objetivo

Este projeto tem como foco o aprendizado de:

* Criação de APIs REST
* Integração com banco de dados
* Boas práticas de organização de código
* Uso do Spring Boot no desenvolvimento backend
